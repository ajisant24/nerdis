Live Audio Coding berbasis Python di Web

1. Cara Menjalankan

  1. Klik Init Audio
  2. Pilih preset (Ambient, Gamelan, Horor, Drum)
  3. Tekan Ctrl + Enter untuk menjalankan
  4. Tekan Ctrl + . untuk stop suara

2. Perintah Dasar

play("C4", 0.5)
instrument("pad")
reverb(3)

3. Daftar Instrumen

instrument("lead")
instrument("bass")
instrument("pad")
instrument("pluck")
instrument("metal")

4. Drum Machine

kick()
snare()
hihat()

5. Contoh Ambient

instrument("pad")
reverb(5)
play("C3",2)
play("G3",2)
play("E3",2)

6. Contoh Gamelan Digital

instrument("pluck")
play("G3",0.3)
play("C4",0.3)
play("E4",0.3)

7. Contoh Horor / Mistis

instrument("metal")
reverb(6)
play("C2",1.5)
play("D#2",1.5)
play("B1",2)

8. Contoh Beat Drum

kick()
hihat()
snare()
hihat()

9. Live Coding

Edit kode saat suara berjalan, lalu tekan Ctrl + Enter lagi untuk mengubah suara secara realtime.

10. Ringkasan Perintah

play(nada, durasi)
instrument("jenis")
reverb(nilai)
kick()
snare()
hihat()
